# `u32`

## Overview
32-bit unsigned integer operations.

Import:
```flavent
use u32
```

## API
- `wrap(x: Int) -> Int`
- `u32And(a: Int, b: Int) -> Int`
- `u32Or(a: Int, b: Int) -> Int`
- `u32Xor(a: Int, b: Int) -> Int`
- `u32Not(a: Int) -> Int`
- `u32Shl(a: Int, n: Int) -> Int`
- `u32Shr(a: Int, n: Int) -> Int`
